Transition

The transition property allows you to change the style of an element smoothly over a specified duration. For example, you can change the background color of a button when a user hovers over it.

Example:

Â¨ 
            
            .button {
                background-color: Blue;
                transition: Background-color 0.5s ease;
                }
                
            .button:hover {
                    background-color: green;
                }

Keyframes animation

Example:

Keyframes allow you to define a series of styles that an element should have at certain points in time. You can then use these keyframes to create an animation.

Example:

            
                @keyframes slidein {
                    from {
                        transform: translateX(-100%);
                    }
                    to {
                    transform: translateX(0);
                    }
                    }
                    
                    .slide {
                        animation: slidein 1s ease;
                    }

Transform

The transform property allows you to modify the position, size, and shape of an element. You can use transforms to create animations such as scaling, rotating, and skewing.

Example

            
                .box {
                    transform: scale(1);
                    transition: transform 0.5s ease;
                }
                
                .box:hover {
                    transform: scale(1.2);
                }

Animation shorthand

- You can use the 'animation' shorthand property to combine multiple animation properties into a single declaration.

Example:

            
                .box {
                    animation: slidein 1s ease;
                }
                
                @keyframes slidein {
                    from {
                        transform: translateX(-100%);
                }
                to {
                    transform: translateX(0);
                }
                }

Information

These are just a few examples of the different ways you can animate elements using CSS. There are many properties and values you can use to create animations.
